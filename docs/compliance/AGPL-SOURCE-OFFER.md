# AGPL Source Offer

Antreva Remote is based on RustDesk OSS and RustDesk Server OSS, which are
licensed under AGPL-3.0. Every distributed Antreva Remote binary must have
corresponding source available under the AGPL-3.0 terms.

## Release Requirement

For each released Windows binary:

1. Publish the exact source revision used to build it, including submodule SHAs.
2. Publish Antreva patches, branding assets, build scripts, and release notes.
3. Keep the AGPL license text and upstream copyright notices.
4. Include a visible source link or source offer in the app/about material and
   next to the download.
5. Keep the source available for as long as the binary is distributed.

## Source Offer Text

Antreva Remote is based on RustDesk OSS and is licensed under AGPL-3.0. You can
obtain the corresponding source code for this build from the Antreva Remote
source repository or release source archive listed next to the download.
